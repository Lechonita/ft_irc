LISTE DES COMMANDES

CHANNEL COMMANDS :
	x JOIN (join channel)
	x PART (leave channel)
	- NAMES (list les utilisateurs de la channel)

 BASIC COMMANDS :
	x PASS (pour rentrer le mdp) OK
	x NICK (change your nick) OK
	x USER (set username and mode) OK
	x PRIVMSG (send a private message to another user)
	x QUIT (quit server)
	- LIST (list channels)
	x WHOIS (get personnal information about a user)

CHANNEL OPERATOR COMMANDS :
	x KICK
	x TOPIC
	x INVITE
	x MODE
